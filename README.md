# 人生地圖 · 智慧農民曆 🏯

2026 丙午年智慧農民曆，結合傳統文化與現代科技。

## ✨ 功能特色
- 📅 正統農民曆查詢（1900-2100年）
- 🧧 每日宜忌、沖煞、五行
- ⛩️ 動態神仙降臨動畫
- 🎋 數位擲筊祈福
- 📱 PWA 支援（可安裝至手機）
- 🏯 CSS 繪製神宮建築

## 🔧 技術棧
- HTML5 / CSS3 / JavaScript
- Tailwind CSS
- lunar-javascript (農民曆核心)
- PWA (Progressive Web App)

## 🌐 線上體驗
https://sethjesus.github.io/zodic/

## 📱 安裝到手機
1. 用 Chrome 開啟網址
2. 點擊「安裝應用程式」提示
3. 即可從主畫面直接開啟
